Blockscribe is an AI-driven middleware that **predicts and explains blockchain transactions in natural language before users sign them**. **Pre-Transaction Analysis**: When a user initiates a transaction, Blockscribe intercepts the payload and sends it through a decoding pipeline powered by Ethers.js and a GPT-based interpretation layer. The system maps the function signature to known contract templates and produces natural-language explanations like:

> "You are about to grant permission for 2.5 ETH to be swapped for approximately 4,000 USDC on Uniswap."

The explanation is displayed through a clean, embeddable React widget that integrates seamlessly into dApps and wallets, **before the user signs the transaction**.

## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ and npm
- Git
- MetaMask or compatible Web3 wallet
- OpenAI API key (for AI features)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/blockscribe.git
   cd blockscribe
   ```

2. **Install dependencies**
   ```bash
   # Frontend
   cd Frontend
   npm install
   
   # Backend
   cd ../Backend
   npm install
   ```

3. **Set up environment variables**
   ```bash
   # Frontend (.env.local)
   VITE_WALLETCONNECT_PROJECT_ID=your_project_id
   VITE_ALCHEMY_ID=your_alchemy_key
   
   # Backend (.env)
   OPENAI_API_KEY=your_openai_key
   MONGODB_URI=mongodb://localhost:27017/blockscribe
   ```

4. **Start development servers**
   ```bash
   # Frontend (http://localhost:5173)
   cd Frontend && npm run dev
   
   # Backend (http://localhost:3001)
   cd Backend && npm run dev
   ```
